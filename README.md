# Agile-Driven E-Commerce Dashboard

A full-stack dashboard for managing orders, built with Java Spring Boot and a lightweight frontend.

## Features
- RESTful API for order management
- Dynamic frontend table rendering
- XML-based configuration
- SQL schema for order tracking

## Setup
1. Clone the repo
2. Run backend: `./mvnw spring-boot:run`
3. Open `frontend/index.html` in browser

## Notes
- Extend controller for inventory and feedback modules
- Use Agile sprints to modularize features
